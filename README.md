1. AsyncStorage hoạt động như thế nào?
-AsyncStorage lưu dữ liệu dạng key–value trong bộ nhớ thiết bị.
-Hoạt động bất đồng bộ (async/await)
-Dữ liệu không mất khi tắt app
2. Vì sao dùng AsyncStorage thay vì state?
-State: chỉ lưu tạm khi app chạy → tắt app là mất
-AsyncStorage: lưu lâu dài → dùng cho token, giỏ hàng, setting
3. So sánh với Context API
AsyncStorage: lưu trữ lâu dài , tốc độc chậm, dùng cho lưu trữ(giống ổ cứng)
Context API: chia sẻ state giữa component,tốc độ nhanh, dùng cho quản lý state (giống RAM)
